Board Legend

This is a Trello-style local board with lane counts, compact cards, brief owner lines only when real agents are involved, and concise files/domains. Treat movement as a manual drag-and-drop-like workflow.

To Do -> Pending / Claimed -> Review -> Done or Blocked

How to Update This Board

  1. Claim from To Do only after Leader assignment or clear authorization.
  2. Move claimed work to Pending / Claimed and add a real owner/agent line plus compact files/domains.
  3. Move completed work to Review with outputs and verification notes.
  4. Leader moves reviewed work to Done; workers do not self-approve.
  5. Use Blocked for missing approvals, source material, runtime access, or sensitive decisions.
Static HTML. No external CDN dependencies. Source board: docs/agent-task-board.md. Related historical tracker: homepage-layout-delivery-whiteboard.md.
